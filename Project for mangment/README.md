# INOVATION GROUP - نظام إدارة النقل والشحن

## نظرة عامة
منصة متكاملة لإدارة خدمات النقل والشحن تتيح للعملاء طلب خدمات النقل، وللمندوبين إدارة الطلبات، وللشركات إدارة عملياتها.

## المميزات
- ✅ نظام تسجيل دخول متعدد الأدوار (عميل - مندوب - شركة)
- ✅ إنشاء وإدارة طلبات النقل
- ✅ تتبع الطلبات في الوقت الفعلي
- ✅ نظام محادثة بين العملاء والمندوبين
- ✅ واجهة مستخدم عربية متجاوبة
- ✅ تخزين بيانات محلي (LocalStorage)
- ✅ إنشاء حسابات حقيقية مع تحقق من البيانات
- ✅ إدارة الملف الشخصي وتغيير كلمة المرور

## الأدوار المتاحة

### 1. العميل
- إنشاء طلبات نقل جديدة
- تتبع حالة الطلبات
- التواصل مع المندوبين
- عرض سجل الطلبات
- إدارة الملف الشخصي

### 2. المندوب
- استقبال الطلبات المتاحة
- إدارة الطلبات المخصصة
- تحديث حالة الطلب
- التواصل مع العملاء
- إدارة الملف الشخصي

### 3. الشركة
- تسجيل بيانات الشركة
- إدارة المندوبين
- تقارير وأحصائيات
- إدارة الملف الشخصي

## كيفية التشغيل

1. قم بتحميل جميع الملفات في مجلد واحد
2. افتح ملف `index.html` في متصفحك
3. الموقع جاهز للاستخدام!

## بيانات تجريبية

يمكنك استخدام البيانات التالية للتجربة:

### عميل تجريبي:
- الهاتف: `+201000000001`
- كلمة المرور: `123456`

### مندوب تجريبي:
- الهاتف: `+201000000002` 
- كلمة المرور: `123456`

## إنشاء حساب حقيقي

1. انتقل إلى صفحة "إنشاء حساب"
2. اختر الدور المناسب (عميل، مندوب، أو شركة)
3. املأ البيانات المطلوبة
4. سيتم إنشاء حسابك ويمكنك تسجيل الدخول فوراً
